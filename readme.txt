Примеры запуска:
python volatility_surface_by_SABR.py mid_p file.csv -p 28NOV20 4DEC20 11DEC20 18DEC20
python get_volatility_and_price_from_SABR.py 17337 17000 2020-12-04:8:00 file.csv

mid_p - put-опцион, цена берется как среднее между ask и bid.
file.csv - имя файла, в который выведется матрица волатильности.
При этом в файл parametres_file.csv выведутся некторые другие переменные, в том числе цена базового актива и параметры поверхности.
28NOV20 4DEC20 11DEC20 18DEC20 - список исходных файлов с информацией с Deribit.
17337 - текущая цена спот
17000 и 2020-12-04:8:00 - страйк и время экспирации, для которых нужно рассчитать новое значение волатильности.
Значения этих входных параметров ограничены существующей поверхностью.
Для текущих данных с Deribit значение страйка должно находиться на отрезке [16000, 19000].

Первый скрипт работает пару секунд, нужно подождать.
В результате работы каждого скрипта в браузере должна открыться новая вкладка с изображением поверхности.
После работы второго скрипта в консоль выведется информация о волатильности и цене опциона при заданных страйке и времени экспирации.
